# 📊 Market OSC Dashboard

A simple, clutter-free financial dashboard built to monitor all four major market indices. 

This tool dynamically adjusts to different timeframes, allowing you to track macro-economic and tactical trends seamlessly. **Completely free to use. Enjoy!** 😊

## ✨ Features

* **Multi-Index Tracking:** Easily switch between the four major market indices using the sidebar dropdown.
* **Smart Timeframes:** Choose between **6-month, 1-year, 3-year, and 5-year** views. 
* **Autonomous Scaling:** 
  * Data intervals adjust automatically based on your selected timeframe.
  * Oscillator (OSC) settings and risk zones dynamically recalculate to fit the period, ensuring relevant overbought/oversold signals.
* **Manual Control:** Fully customize and override the OSC settings and risk zones manually via the left panel if you prefer your own parameters.
* **Relative Strength Overlay:** Toggle and overlay the relative strength of either the **S&P 500** or the **Nasdaq 100** for advanced comparison.
  
## 🛠️ How to Use

1. Use the **left sidebar panel** to select your target index and desired timeframe.
2. Observe the automated technical indicator adjustments, or use the manual inputs to fine-tune the settings.
3. Use the overlay toggle to analyze relative strength against major benchmarks.
